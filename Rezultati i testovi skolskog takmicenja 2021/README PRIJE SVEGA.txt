Zadatke 2 i 4 testirajte na HackerRank, pošto su odatle zadaci. Zovu se "Strong password" i "3D surface area" respektivno.

Zadatke 1 i 3 testirajte na način:
1) Otvorite program "Input i output iz fajla framework" i copy pasteajte algoritamski dio vašeg zadatka u predviđeni prostor za njega. 
2) Onda zamijenite svaki input u vašem kodu sa file1, a output sa file2.
3) Postavite ispravne path-ove, file1 treba imati path koji odgovara FOLDERU u kojem se nalaze input fileovi za taj zadatak (Folder se zove z1 testovi za zadatak -> input npr) a file2 treba u neki drugi folder koji sami napravite.
4) Pokrenite program, on će za svaki input iz foldera input generisati output u vaš folder 

5) Otvoriti autotest program, postaviti jedan path da odgovara folderu kojeg ste maloprije napravili, u kojem se nalaze output fileovi za vaš zadatak. Drugi path treba da odgovara folderu u kojem se nalaze ispravni outputi (Folder se zove z1 testovi za zadatak -> output npr)
6) Kad pokrenete program, ispisat će vam koliko vam je ispravnih testova (output.txt vaš se podudara skroz sa ispravnim kojeg sam vam ja poslao)
